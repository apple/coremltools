Core ML Tools API OverviewÔÉÅ

Use the Core ML Tools API to convert deep learning models and neural networks to Core ML. For details, see the Core ML Tools API Reference.

For example, you can use the Unified Conversion API (convert()) to convert TensorFlow and PyTorch source model frameworks to Core ML. For the conversion parameters, see the convert() method.
