Normalizer

A normalization preprocessor.

message Normalizer {
    enum NormType {
        LMax = 0;
        L1 = 1;
        L2 = 2;
    }

    NormType normType = 1;
}

Normalizer.NormType

There are three normalization modes, which have the corresponding formulas:

Max
\[max(x_i)\]
L1
\[z = ||x||_1 = \sum_{i=1}^{n} |x_i|\]
L2
\[z = ||x||_2 = \sqrt{\sum_{i=1}^{n} x_i^2}\]
enum NormType {
    LMax = 0;
    L1 = 1;
    L2 = 2;
}
