Converting Deep Learning Models
===============================

Use the Core ML Tools Unified Conversion API, coremltools.convert() (available from version 4.0 onwards) to convert deep learning models to the CoreML model format, to be deployed with the Core ML framework. You can convert source models from the `PyTorch <https://pytorch.org>`_ and `TensorFlow <https://www.tensorflow.org>`_ frameworks.
